# Backlink and Brand Opportunity

Created: 2026-07-09

## Summary

YTgify still receives useful search traffic and backlinks even after removal from the Chrome Web Store. The homepage should keep matching "YouTube to GIF" search intent, but it should also make Jeremy Watt, neonwatty.com, GitHub, Bleep That, and Seatify much more visible.

The goal is not to turn this into a new product pivot today. The goal is to make the existing traffic work harder for personal branding, backlinks, project discovery, and the open-source story.

## Known Signals

Google Search Console snapshot from the domain property:

- Last 3 months through 2026-07-06: 537 clicks and 14.7k impressions.
- The homepage gets essentially all clicks: 537 clicks and 14,388 impressions.
- Blog pages have impressions but very few or no clicks.
- External links: 38 total, all to the homepage.
- Top linking sites: reddit.com, chrome-stats.com, and neonwatty.com.
- Strong query clusters: `youtube to gif`, `yt to gif`, `youtube to gif converter`, `youtube video to gif`, `youtube to gif no watermark`.
- Branded query: `ytgify` ranks at position 1.0 with strong CTR.

## Positioning

Keep:

- YouTube to GIF language for SEO continuity.
- The removal notice, because it is honest and explains the current state.
- The legacy download, for users who still came looking for it.

Change:

- Move the legacy install flow out of the first interaction.
- Make "Built by Jeremy Watt" a first-class homepage section.
- Add clear links to neonwatty.com, GitHub, Bleep That, and Seatify.
- Link to the existing neonwatty.com YTgify build note/case-study page.
- Reframe YTgify as an open-source experiment and proof of product taste, not only as a removed extension.

## First Implementation Pass

1. Split the removal notice from the local install instructions.
2. Keep the removal notice near the top.
3. Move local install below the brand/current-work sections.
4. Add a "Built by Jeremy Watt" section with direct links to build notes and source code.
5. Add a compact "More from Jeremy" project link section.
6. Update footer links to include Bleep That and Seatify.
7. Adjust metadata enough to include Jeremy/open-source branding while preserving the core search terms.
